# TorchScript Weights

Place your TorchScript demo weights here so the AOI page can find them without additional configuration. For packaging, weights under `public/models` are also discovered; `assets/models` is convenient during dev.

- CPU: `aoi_cpu.ts`
- GPU: `aoi_gpu.ts`

Both filenames are discovered automatically by the backend; you can also override the paths in the AOI UI.
